# Creating Desktop Shortcut

To create a convenient desktop shortcut for your FarmWise Agricultural AI application:

## Method 1: Automatic (Copy-Paste)

1. Right-click on `START_PROJECT.bat`
2. Select "Send to" â†’ "Desktop (create shortcut)"
3. Rename the shortcut to "FarmWise AI"
4. Done! Double-click the desktop icon to start the app

## Method 2: Manual

1. Right-click on your Desktop
2. Select "New" â†’ "Shortcut"
3. In the location field, paste:
   ```
   d:\cap\leaf-disease-proj\farmwise-agricultural-ai\START_PROJECT.bat
   ```
4. Click "Next"
5. Name it "FarmWise Agricultural AI"
6. Click "Finish"

## Optional: Add a Custom Icon

1. Right-click on the shortcut
2. Select "Properties"
3. Click "Change Icon"
4. Browse and select an icon (you can use any .ico file)
5. Click "OK"
